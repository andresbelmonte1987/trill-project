Readme para Trill Digital Media

Este es un sitio web creado para Trill Digital Media, una agencia digital que se especializa en transformación digital. Ofrecen servicios como creación de sitios web, gestión de redes sociales y campañas publicitarias, así como recursos gratuitos en su sitio web sobre marketing digital.

Características
Diseño responsive y moderno utilizando Bootstrap 5.1.3
Página principal con carrusel de imágenes y menú de navegación
Secciones de Servicios, Consultoría, Sobre Nosotros y Contacto
Contenido informativo sobre la agencia y el equipo remoto de digital nomads
Recursos gratuitos de marketing digital en el sitio web

Cómo utilizar
Puedes clonar o descargar los archivos del sitio web y abrir el archivo index.html en un navegador web para ver el sitio web en acción.
